# InciScan - Real-Time Incident Analysis System

## Overview
InciScan is a real-time incident analysis system designed to detect and analyze incidents such as crowd density, suspicious activities, violence, and more using advanced ML algorithms and live video feeds.

## Tech Stack

### Frontend
- **Framework**: React with TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Context API
- **Real-time**: WebSocket / Socket.io Client
- **Maps**: Leaflet / Mapbox
- **Charts**: Chart.js

### Backend
- **Runtime**: Node.js
- **Framework**: Express with TypeScript
- **Real-time**: Socket.io
- **Database**: PostgreSQL (PostGIS) + Redis
- **Auth**: JWT

## getting Started

### Client
```bash
cd client
npm install
npm run dev
```

### Server
```bash
cd server
npm install
npm run dev
```
